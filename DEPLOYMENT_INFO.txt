ДЕПЛОЙ НА СЕРВЕР:
------------------
Вариант 1 (только готовые файлы):
  Скопируй содержимое корневой папки (assets/, documents/, fonts/, 
  images/, index.html, robots.txt) на веб-сервер

Вариант 2 (полный проект):
  1. Скопируй всю папку на сервер
  2. Запусти: npm install
  3. Для разработки: npm run dev
  4. Для продакшена: npm run build

ВАЖНО:
------
- Не копируй node_modules (установится через npm install)
- Проверь .env.local перед деплоем
- API настроен на https://tourderussie.ru/api
